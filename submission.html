5. Deeper Understanding Statement

Two deeper understanding features were implemented. First, a body colour picker was added to the Bamborghini Baventador model, allowing users to change the car's paint colour in real time using Three.js's scene.traverse() method to target specific named meshes exported from Blender. Second, a custom PHP/SQLite REST API was developed to track how many times each car model has been viewed, with live view counts displayed in the info panel via asynchronous JavaScript fetch calls.
